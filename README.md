# Morpheus

## 中文

Morpheus 是一个本地优先的个人操作空间，用来整理念头、项目、日常记录、提醒、健康线索和 AI 对话。

它更像一个给个人使用的工作台：你可以把零散想法先放进来，再逐步沉淀成定念、项目、日志或行动；也可以让 AI 帮你回看上下文、整理结构、生成下一步。

当前公开版本是 Web 客户端，适合在本机运行和体验核心界面。

### 一行启动

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/gaoyiiiii/Morpheus/main/scripts/install-and-run-web.sh)"
```

### 本地运行

```bash
npm install
npm run build
npm start
```

然后打开：

```text
http://127.0.0.1:2199/
```

## English

Morpheus is a local-first personal operating workspace for thoughts, projects, daily notes, reminders, health context, and AI conversations.

It is designed as a personal workbench: capture loose ideas, turn them into durable thoughts, projects, logs, or actions, and let AI help you review context, organize structure, and find the next step.

This public version is the Web client and can be run locally to try the core experience.

## One-Line Start

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/gaoyiiiii/Morpheus/main/scripts/install-and-run-web.sh)"
```

## Local Run

